# CIC Game Development Program Website

## Project Description
This website was created to promote the Game Development program at the College of the Immaculate Conception. The website introduces the program, highlights the learning environment, and shows the skills students can gain from studying game development.

## Features
- Responsive layout using Bootstrap
- Hero section introducing the program
- Information sections about the learning environment and what will the students will learn
- Footer with school information and social media links

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Visual Studio Code

## Project Structure
/assets
index.html
style.css
README.md

1. Open `index.html` in any web browser.
