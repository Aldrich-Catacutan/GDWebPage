Why Join This Course?

  • Learn the basics of game design and game programming.
  • Develop creativity, problem-solving, and logical thinking skills.
  • Gain hands-on experience in creating simple digital games.
  • Understand how games are planned, designed, and built.
  • Build foundational skills useful for IT and creative technology courses.

Skills Gained

  • Basic game design and development concepts.
  • Fundamental programming and coding skills.
  • Problem-solving and logical thinking.
  • Creativity and idea development.
  • Debugging and basic testing skills.
  • Teamwork and communication during project work.

Tools Used

  • GDevelop – for creating 2D games.
  • Visual Studio Code – for coding and scripting.
  • Piskel / Pixel Art tools – for simple game assets.

Learning Environment

The Mac Laboratory provides a structured learning environment equipped with computers and essential software used in the Game Development course.

It supports hands-on activities where students practice designing, coding, and testing games under guided instruction, enhancing both technical skills and practical understanding.
